A small data engineering project built with Python and MongoDB.

It demonstrates how raw expense data can be ingested, transformed, and analyzed using an ELT-style workflow.

## What it does

- Stores expense events as raw data in MongoDB
- Builds aggregated analytics tables with Python
- Analyzes and visualizes the data in a Jupyter notebook

## Data

- expenses: raw transaction data
- category_summary: spending per category
- monthly_summary: spending per month

## Tech
